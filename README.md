# MAUSAM, Weather App

Live Demo Link : https://umer-exe.github.io/weather-app-mausam/

A clean, responsive weather app built with HTML, CSS, and JavaScript. Search cities, use your current location, switch between °C and °F, and view tidy weather cards.

- City Search, add any city by name
- Use My Location, quick geolocation lookup
- Temperature Units, toggle °C and °F
- Refresh and Clear, update data or remove cards instantly
- Responsive Design, scales well on desktop and mobile
- Modern UI/UX, clean cards with soft shadows and rounded corners, smooth interactions
- Footer Social Links, LinkedIn and GitHub icons via CSS masks, perfectly aligned, open in new tabs
- Pure Frontend, HTML, CSS, vanilla JavaScript, no build tools, no external JS dependencies
- Powered by OpenWeather, geocoding and current weather APIs
- Deployed on GitHub Pages, easy to share and test
